# Lumina Cosmic Variance

Analysis of halo and galaxy cosmic variance in the Lumina and Lumina-EDE simulations.

The initial target is the halo mass function at redshift z approximately 12 using
non-overlapping mock fields and `Group_M_Crit200`.

## Layout

- `notebooks/`: exploratory analysis
- `src/lumina_cv/`: reusable analysis code
- `tests/`: automated validation
- `configs/`: analysis configurations
- `scripts/`: executable analysis scripts
- `slurm/`: cluster job scripts
- `figures/`: selected figures

Simulation data and generated outputs are not stored in this repository.
